# Arnav Tomar — Projects

## Project 1: Sentiment Intelligence Engine

**Year**: 2026  
**Type**: Full-Stack AI / ML Platform  
**Status**: Production  
**Live Demo**: https://sentiment-intelligence-engine.vercel.app  
**GitHub**: https://github.com/arnavtomar18/Sentiment-Intelligence-Engine

### Description
A full-stack AI/ML platform covering 5 real-world domains — News (Fake/Real Detection), Hotel Reviews, Fashion, App Reviews, and OTT Content Recommendation. 18+ trained ML models unified under a single production system.

### Tech Stack
Python, FastAPI, React 18, Vite, Scikit-learn, XGBoost, LightGBM, TF-IDF

### Key Highlights
- 18+ trained machine learning models across 5 distinct domains
- Soft-voting ensemble combining Logistic Regression, SVC, XGBoost, and LightGBM
- TF-IDF cosine similarity engine powering real-time OTT content recommendation
- FastAPI backend with domain-specific routers and batch prediction endpoints
- Frontend deployed on Vercel using React 18 + Vite
- One of the most technically ambitious projects built in a first year of college

## Project 3: Sentiment Analysis

**Year**: 2025  
**Type**: NLP / Data Science  
**GitHub**: https://github.com/arnavtomar18/Sentiment-Analysis

### Description
An NLP pipeline to classify tweets as Positive, Negative, or Neutral. Arnav's foundational NLP project that led directly to the Sentiment Intelligence Engine.

### Tech Stack
Python, NLP, Scikit-learn, Matplotlib, Seaborn

### Key Highlights
- Full preprocessing pipeline: tokenization, stopword removal, TF-IDF vectorization
- Multiple classification models trained and evaluated with Scikit-learn
- Sentiment trends and model performance visualized using Matplotlib and Seaborn
- Served as the foundation for Arnav's deeper ML and NLP specialization
